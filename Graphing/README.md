# Graphing different equation

In this folder, I am trying to graph some equation in a plane

to run the main file with is (Making_graph.py) run the following commands

```bash
# Low quality (480p)
manim -pql Making_graph.py Graph

# Medium quality (720p)
manim -pqm Making_graph.py Graph

# High quality (1080p)
manim -pqh Making_graph.py Graph

# 4K quality (2160p)
manim -pqk Making_graph.py Graph
